Metakit sample code - Dbf2mk


This is a generic utility for the Metakit library.

What it does: Dbf2mk is a utility program which reads data from one or more industry-standard DBF files and saves the data in a Metakit datafile.

Known problems: None. This utility has not yet been extensively tested. This version can handle files of any size, but it does not load DBase "memo" fields and ignores all index files and calculated fields.
